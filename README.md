# Ista421_final
Final Project For Ista 421

This project analyzes the NYC Taxi Trip Record Data to explore patterns in taxi fares and build machine learning models to better understand pricing behavior.

The goal is to investigate potential bias in fare pricing based on trip characteristics (distance, time, passenger count, location) and in turn help prediction with fares.

Objectives

- Perform exploratory data analysis on taxi trip data
- Identify key factors influencing fare pricing
- Detect potential bias/patterns in pricing
- Build + Evaluate machine learning models
- Communicate insights through visualizations
