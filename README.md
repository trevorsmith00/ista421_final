# ISTA 421 Final Project — NYC Taxi Fare Analysis
**Trevor Smith | Professor Ashton | ISTA 421**

This project analyzes the NYC Yellow Taxi Trip Record Data (January 2026) to investigate whether machine learning models reveal systematic biases or inconsistencies in fare pricing across geographic locations and trip characteristics.

## Research Question
Do machine learning models used to predict NYC taxi fares reveal systematic biases or inconsistencies across geographic locations and trip characteristics, and which predictors contribute most to these disparities?

Python 3.9+ is recommended.

## How to Run
1. Download `yellow_tripdata_2026-01.parquet` from D2L and place it at `data/raw/yellow_tripdata_2026-01.parquet`
2. Open `notebooks/taxi_analysis.ipynb` in Jupyter or VS Code
3. Run all cells top-to-bottom (**Kernel → Restart & Run All**)

The notebook is fully self-contained. All figures are saved automatically to `figures/`.

## Notebook Sections
| Section | Contents |
|---|---|
| 1 — Data Exploration | Summary statistics, correlation heatmap, feature discussion |
| 2 — Data Cleaning | Remove outliers, handle NaN, standardize features |
| 3 — Visualizations | Fare/distance histograms, scatter plot, boxplot by zone |
| 4 — Linear Regression | Normal equations from scratch, Ridge CV tuning, sklearn verification |
| 5 — Decision Tree | GridSearchCV over max_depth/min_samples_leaf, feature importance, residual maps |
| 6 — Conclusions | Hypothesis assessment, pricing fairness discussion, limitations |

## Dataset
Source: NYC Taxi & Limousine Commission (TLC)
- 4,181,444 observations, 20 features
- Download: https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page
- **The dataset is submitted separately on D2L and is not included in this repository.**
